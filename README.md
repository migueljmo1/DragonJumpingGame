# Dragon Jumping Game

Juego 2D desarrollado en Godot inspirado en el juego del dinosaurio de Chrome.

## Características
- Saltar y agacharse.
- Enemigos con movimiento senoidal.
- Sistema de monedas.
- Revivir con anuncios.
- Optimizado para dispositivos móviles.

## Tecnologías
- Godot Engine
- GDScript
